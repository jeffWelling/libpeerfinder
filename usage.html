Usage

 

There are two primary ways of using libpeerfinder.  You can execute it from the command line, like this;

 $ ./libpeerfinder -tracker

And it would return a ("\n" seperated) list of IPs that you could query for torrents,
Or you could use

 $ ./libpeerfinder -torrentdht

And it would return a list of IPs that are known to host DHTs for torrents.  For a complete list of the plugins available to your instance of libpeerfinder, use `./libpeerfinder -h` .  The expected output from the above two examples is in the format of;

 $ ./libpeerfinder -torrentdht

 192.168.0.1

192.168.0.2

192.168.0.3

192.167.3.4

...


The altnerative to using it from the command line would be to hook into the library itself.  It's interface is published here on rdoc.info .
