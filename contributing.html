Contributing

 

If you'd like to contribute ideas or to discussions, check out the google group we have setup here.

To contribute to the codebase, you can either join Github and fork the project, or you could clone it and submit patches to me/us via email.

Obviously, this is a sad excuse for a website, we accept contributions towards that as well, via the gh-pages branch.
