libpeerfinder by jeffWelling

libpeerfinder is a library which will locate peers, eliminating the need for bootstrapping nodes in decentralized peer-to-peer networks.

libpeerfinder is intended to be the solution to the boostrapping node problem. The problem wherein a decentralized system needs an initial seed address in order to be able to connect with the rest of the decentralized swarm on first contact, creating a single point of failure.
